The goal of the challenge is to reconstruct a flag from a pseudo-random character string generated by a custom algorithm implemented by the provided binary :

```python
for i in range(len(flag)):
    enc_flag += flag[i]
    enc_flag += gen_random(len(flag) - i)
```

So the code to reconstruct the flag is as follows:
```python
flag = b''
for i in range(flag_len):
    flag += p8(enc_flag[0])
    enc_flag = enc_flag[flag_len - i + 1:]
```
